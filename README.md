# minigrep-rust
Rust minigrep exercise

### Install Rust
Use rustup

[https://rustup.rs/]

### Setting up Rust in VSCode
Install RLS extension in VSCode

[https://github.com/rust-lang/rls-vscode]

### Building and Running

#### Build
Dev:
```
$ cargo build
```

Release:
```
$ cargo build --release
```

#### Test

```
$ cargo test
```

#### Run

```
$ cargo run

$ cargo run to poem.txt
```
